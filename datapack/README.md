If you do not apply [this resource pack](https://github.com/minijun09/ochak-brdatapack/releases), items may appear broken.

---

## Datapack in contains custom item
- healing
    * first aid kit
    * bandage
    * energy drink
    * adrenaline
    * painkiller
- weapons
    * revolver single action
        + reloading
        + magazine
        + recoil
- ammo
    * .357 magnum
- throwable
    * impact grenade
    * smoke grenade
- user interface
    * remaining bullets display
- some **unknown** items
